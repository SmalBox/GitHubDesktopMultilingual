# GitHubDesktopMultilingual
GitHubDesktop multilingual 多语言切换软件(默认为汉化)。

## 简介
   
   - GitHubDesktop 官方没有支持多语言切换功能。也找过其他做汉化的现成方案，基本都是跟这软件版本针对性的做汉化再发出来。由于这种方式的工作量巨大，现在基本上已经很难有人能坚持跟着版本坚持更新了。
   - 为了将多语言方式做的相对通用一些，减少更新带来的巨量多语言翻译工作。能方便的使GitHubDesktop实现多语言。写了此软件来帮助快速切换、备份、恢复。
   - 软件使用多语言字典来配置多语言的切换，可以根据多语言字典翻译成其他语言，并再表头加上 @SmalBox 即可对该语言进行切换。
   - 欢迎各位为其填充各个其他的语言、完善多语言字典，让GitHubDesktop更加顺手。

## 使用

   - 使用发布的包(推荐)
      - 到 Release 中下载最新的版本。
      - 解压压缩包，(默认为切换为简体中文)直接运行 GitHubDesktopMultilingual 文件。
      - 根据提示选择语言切换(语言切换时会自动备份配置文件到当前目录的Backup中) 或 恢复备份(根据当前目录的Backup中的配置文件恢复到软件中)
      - 如果要切换其他语言 或 自行添加切换其他语言：
         - 打开配置文件 GitHubDesktopReplaceDic.csv 文件
         - 根据已有的多语言对照表，可添加其他语言或增加更多的行来提升多语言覆盖能力。
         - 语言表头后加上 @SmalBox 即可设置成切换的目标语言。
   - 运行源码
      - clone 本项目。
      - 其中 Multilingual.py 为主入口文件
      - 本项目中使用 Python 版本为 3.9
      - 涉及到的包 pywin32
      - 安装所涉及到的包后，可用 python 命令直接运行 Multilingual.py 即可运行本软件。
      - 配置文件等其他操作参考 “使用发布的包” 中的具体操作。

- V1.0.0
- 添加自动切换语言的脚本软件(当前仅支持Windows)。
- 软件默认为汉化。
- 主要提供两个功能：English切换其他语言(默认为简体中文)、还原English。
- 多语言字典文件 GitHubDesktopReplaceDic.csv 中提供部分中文汉化项。
- 多语言字典中可自行添加列对照已有格式翻译成其他语言。
- 多语言字典中在要切换的语言表头后加上 @SmalBox 即可设置成切换的目标语言。
   - 例如：简体中文@SmalBox
